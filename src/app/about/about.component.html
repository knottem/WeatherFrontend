About

This project, a comprehensive weather forecasting application, is being developed as part of my thesis at Nackademin. The application, designed to blend accuracy with user-friendliness, integrates data from multiple trusted sources to deliver real-time weather updates.

The frontend of the application is built using Angular, known for its robustness and efficiency in creating dynamic web pages, along with Tailwind CSS for its flexible and responsive design capabilities. This combination allows for an intuitive and user-friendly interface that is accessible on a variety of devices and screen sizes.

On the backend, I'm utilizing Java with Spring Boot. The backend is responsible for fetching and processing weather data from SMHI and YR, two reputable meteorological services. By combining data from these sources, the application ensures a more comprehensive and accurate weather forecast.

The Weather images used in the application are sourced from Yr Weather Symbols, contributing to the visual appeal and easy interpretation of weather conditions.
